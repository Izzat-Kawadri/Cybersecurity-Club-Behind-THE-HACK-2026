# 🔐 Cybersecurity Club Event - Behind THE HACK 2026

<p>
 <img src="/images/Image.jpg" alt="main"  >

</p>

This repository documents the **Cybersecurity Behind The HACK 2026 Event** organized by the **Cyber Security Club**. The event aimed to educate participants about essential cybersecurity concepts through interactive and practical learning stations.

## 🎯 Objective

The goal of this event was to raise awareness about cybersecurity risks and best practices by introducing participants to different domains within the cybersecurity field.

## 🧩 Event Structure

The event was divided into **8 interactive corners**, each focusing on a specific area of cybersecurity.
Each corner included:

* 🎬 **Live demonstrations** of cybersecurity tools used for protection
* ⚠️ **Awareness of common hacking tools and techniques** (for educational purposes)
* 🧪 **Hands-on activities** to engage participants
